## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```
* xcode

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/your_username_/Project-Name.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
4. Start server
   ```
   npm start
   ```
5. Run on iOS simulator
   ```
   npm run ios
   ```
